
# She-Balance

She-Balance is a health and wellness application designed to support women's physical and mental well-being. The system enables users to track health-related data, analyze patterns, and receive insights that help maintain a balanced lifestyle.

## Features

* Health tracking (such as menstrual cycle and symptoms)
* Data analysis and visualization
* Personalized wellness recommendations
* Reminder and notification system
* User-friendly interface

## Installation

1. Clone the repository:
   git clone https://github.com/nimmidhas/She-Balance.git
   cd She-Balance

2. Install dependencies:
   pip install -r requirements.txt

## Execution

Run the application using the following command:

python main.py

If the project uses a FastAPI server:

uvicorn main:app --reload

For frontend (if applicable):

cd frontend
npm install
npm start

## Usage

After running the application, open a web browser and navigate to:

http://127.0.0.1:8000

Users can input their health data, track patterns, and view generated insights and recommendations.
